A test package with simple math operations.

# Installation

```
npm install notomath
```

# Usage

### Addition:

```
const ans = add(1, 1)
console.log(ans) // 2
```

### Subtraction:

```
const ans = sub(5, 2)
console.log(ans) // 3
```

### Multiplication:

```
const ans = mul(2, 3)
console.log(ans) // 6
```

### Division:

```
const ans = div(8, 2)
console.log(ans) // 4
```
